## Introduction
Due to the size of our dataset and copyright concerns, we only offer a sample of it. This sample is randomly sampled in our PersonaMov dataset. The movie is called Taxi Driver which is very popular all around the world, directed by Martin Scorsese and starred by Robert Deniro.
### Structure of This Sample
- The JSON file contains the dialogues information with personality and relationshio annotation
- The .mp4 file is clipped according to the timestamp in JSON file.
- The .mp3 file is extracted from .mp4 file using FFmpeg.
